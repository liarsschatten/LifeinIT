# Stichpunkte zum ESP32
- ist ein Mikrocontroller
- Der Controller ist Wifi fähig heißt man kann damit man ein Cyber-physisches System aufbauen
- Der Mikrocontroller ist gegen verschiedene Außeneinflüsse geschützt
- Hat eine integrierte Batterie
- Hat ein OLED Display um Informationen anzuzeigen
- Er wird mit einem integrierten Mikro USB Port (CP2102) angesprochen
- Er wird über die Arduino IDE Programmiert
- Sehr Leistungseffizient

# Textteil
Der ESP 32 ist ein Wifi fähiger Mikrocontroller welcher über die Arduino IDE programmiert werden kann. Er ist gegen ESD und andere schadende Einflüsse geschützt. Dank eigenem OLED Display, Wifi Fähigkeit und integrierter Batterie ist er vielseitig einsetzbar. Die Programmierung wird über einen Mikro USB Port an den Mikrocontroller weitergegeben. darüber wird auch die integrierte Batterie aufgeladen.
