# Allgemeines
Powershell Scripte enden in in *.ps1*. Sie sind ein essentielles Werkzeug zur Administration. Das ist vor allem wichtig für die Automatisation von Prozessen und um Aufgaben die in einem geringen Zeitfenster zu lösen sind schnell zu erledigen. Powershell Scripte müssen als Administrator ausgeführt werden.
# Syntax von Powershell
Die Scripte werden Line für Line ausgeführt, die 
- "#" signalisiert einen Kommentar
- "$" steht vor einer Variable
- "@()" ist eine Liste die innerhalb der Klammern Steht und mit Kommas getrennt wird
