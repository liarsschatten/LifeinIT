# Voll-Backup
Ein Vollbackup ist eine vollständige Sicherung **aller** ausgewählten Daten zu einem bestimmten Zeitpunkt. Das komplette System wird gesichert. Alle Dateien, alles, egal ob sich was geändert hat oder nicht. Kostet viel Zeit und Speicher.

Es bildet die Basis für alle anderen Backup-Typen.
# Differentielles-Backup

- Speichert die Veränderungen seit dem letzten Voll-Backup

# Inkrementelles-Backup

- Speichert alle Veränderungen seit dem letzten Backup
